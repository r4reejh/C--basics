# Prerequisites for writing the nodejs library

1. Nodejs
2. basic concepts:
    - types, scoping
    - functions
    - event loop
    - promises (reject/resolve) ; async/await
    - events
    - modules / npm
3. Typescript, advantages of typescript over Javascript
4. Familiarity with Unit testing
5. Unit testing frameworks in Nodejs
